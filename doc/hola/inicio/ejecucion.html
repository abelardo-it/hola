Inicio - Ejecución

Ejecución

Antes de seguir, asegúrese de instalar el software necesario para la instalación y ejecución de este proyecto.

Estas son los pasos que debe seguir para ejecutar este proyecto:

  1. Acceda a la carpeta donde instaló el proyecto;
  2. Si está en Linux asegúrese que para cualquier intanacia de MySQL y de Apache con los siguientes comandos: 'service mysql stop' y 'sudo service apache2 stop'
  3. Verá que el proyecot tiene tres carpetas: doc, docker y src. Teclee el comando "cd docker";
  4. Teclee el comando "docker-compose up";
  5. Cuando la pantalla haya dejado de imprimir mensajes, abra un navegador y teclee "http://localhost";

Introduzca los datos necesarios para acceder a la aplicación. Estos pasos están indicados en el siguiente paso.
